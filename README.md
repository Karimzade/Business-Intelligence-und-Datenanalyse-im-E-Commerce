# Business Intelligence und Datenanalyse im E-Commerce

## Projektbeschreibung

Dieses Projekt konzentriert sich auf die Analyse von E-Commerce-Daten mithilfe von Excel, SQL und Power BI.

Ziel des Projekts ist die Entwicklung eines professionellen Business-Intelligence-Dashboards zur Analyse von:

- Umsatzentwicklung
- Kundenverhalten
- Produktperformance
- Regionale Verkaufsdaten
- Geschäftliche KPIs

## Verwendete Technologien

- Excel
- SQL
- Power BI
- GitHub

## Projektstruktur

- Datenbereinigung und -aufbereitung
- SQL-Abfragen und KPI-Analysen
- Erstellung interaktiver Dashboards
- Dokumentation geschäftlicher Erkenntnisse
