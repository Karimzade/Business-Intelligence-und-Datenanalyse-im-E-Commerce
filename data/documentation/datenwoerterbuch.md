# Datenwörterbuch

## Datensatzbeschreibung

Der Datensatz enthält Verkaufsdaten eines E-Commerce-Unternehmens.

---

## Tabellenübersicht

| Spalte | Beschreibung | Datentyp |
|---|---|---|
| Row ID | Eindeutige ID jeder Bestellung | Integer |
| Order ID | Bestellnummer | Text |
| Order Date | Bestelldatum | Datum |
| Ship Date | Versanddatum | Datum |
| Customer ID | Kundennummer | Text |
| Customer Name | Kundenname | Text |
| Segment | Kundensegment | Text |
| Country | Land | Text |
| City | Stadt | Text |
| State | Bundesland | Text |
| Region | Verkaufsregion | Text |
| Product ID | Produktnummer | Text |
| Category | Produktkategorie | Text |
| Sub-Category | Unterkategorie | Text |
| Product Name | Produktname | Text |
| Sales | Umsatz | Dezimal |
| Quantity | Verkaufsmenge | Integer |
| Discount | Rabatt | Dezimal |
| Profit | Gewinn | Dezimal |

---

## Erste Beobachtungen

- Der Datensatz enthält Verkaufs-, Kunden- und Produktinformationen.
- Die Daten eignen sich für Umsatz-, Kunden- und KPI-Analysen.
- Besonders relevant sind die Spalten Sales, Profit, Region und Category.
